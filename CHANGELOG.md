# Change Log

All notable changes to the "terraform-visualizer" extension will be documented in this file.

## [1.3.0] - 2026-03-16

### Changed
- Complete redesign of the differences and changes UI
- Enhanced visual differentiation for attribute changes
- Improved color coding with background highlights (red for deletions, green for additions)
- Better value transition display with clearer arrows
- Sensitive data detection and automatic masking
- Improved resource cards with better visual hierarchy

### Added
- Collapsible unchanged attributes section with count badges
- Enhanced theme support with VS Code integration (dark, light, high contrast)

## [1.2.0] - 2025-03-16

### Changed
- Differences and changes UI reviewed

## [1.1.0] - 2025-03-25

### Added
- Full-text search bar for filtering resources
- Modern pill-shaped filter chips for action types
- Expand all / Collapse all batch controls
- Summary dashboard with action counts

### Changed
- Redesigned action labels with color-coded badges
- Improved filter UI with interactive chips
